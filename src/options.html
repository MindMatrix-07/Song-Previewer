Apple/iTunes is used first. If Apple has no preview, the extension searches YouTube with your API key.

How to create the key

  1. Open Google Cloud Console and create or select a project.
  2. Enable YouTube Data API v3 for that project.
  3. Go to APIs & Services, then Credentials.
  4. Create an API key.
  5. Paste the key above and save.
